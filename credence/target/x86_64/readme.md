# x86_64

## generator.h

#### Primary `Emitter` types and `Inserter` object types for x86_64 code generation

## assembly.h

#### Raw ISA mnemonics types, registers, operand storage helpers with string overloading

## stack.h

#### Pushdown stack that represents the x86_64 stack with memory-safe offset resolution

## syscall.h

#### Kernel syscall tables for the Linux and BSD x86_64 architectures

## runtime.h

#### Standard library helpers and compile-time type-safe invocation methods

## memory.h

#### Memory accessors, mediators for memory-safe access during code generation
