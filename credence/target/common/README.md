# Common

This directory contains architecture-agnostic code shared between x86_64 and ARM64 code generators.

## Files

- **flags.h** - Instruction-level flags (Address, Indirect, Align, etc.)
- **memory.h** - Memory access type definitions and predicates
- **stack_frame.h** - Stack frame tracking for function calls
- **runtime.h** - Standard library and runtime function interfaces
- **types.h** - Common data types for code generation
